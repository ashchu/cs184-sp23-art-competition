Caption

Mathematician / Subdividing - subdivide / O, chocolate wolf!

How is the artwork created?

I created this wolf mesh in Blender, and then rendered it using Project 2's meshedit. The main base I used as a building block for the mesh was the icosphere. I built up the mesh by first crafting an icosphere for the head, the torso, the tail, and a single leg. Then, via merging vertices, edges, and faces, along with "symmetrize", I combined these meshes together to form the overall wolf-looking mesh. I also referred to some references of wolves standing to figure out how to best present the wolf's legs in a natural-looking fashion - hopefully I got close to the mark! I thought about creating a mesh of a wolf after deciding that I should leave the possibility of a dragon mesh for another day... So this wolf is inspired by the wolves in the movie: Princess Mononoke!

How long did it take?

Between 5-8 hours or so.

J

Caption

Zuuuubaatt!!

How is the artwork created?

For our art submission, we created a 3D model of Zubat (a Pokemon) using Blender. We mostly traced reference images from Google by aligning vertices to the shapes in the pictures. We also used several subdivision schemes within Blender across the regular polygons to make the mesh more fine-grained. The most important tool, however, was the proportional editing tool. This allowed us to mold any extrusions easily and more smoothly, giving the object a cleaner look. Finally, we upsampled some more using our meshedit program and added a shader to get the complete model.

How long did it take?

6 - 7 hours
